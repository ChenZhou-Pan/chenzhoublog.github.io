## Next Steps

- Create new posts under `content/posts/` using `hugo new posts/post-name.md`
- Choose a theme from the [Hugo Themes](https://themes.gohugo.io/), for example:

  ```bash
  git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
  ```

  Then set `theme = "ananke"` in `hugo.toml`.
