## 下一步

- 在 `content/posts/` 下用 `hugo new posts/文章名.md` 创建新文章
- 在 [Hugo 主题列表](https://themes.gohugo.io/) 选一个主题，例如：

  ```bash
  git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
  ```

  然后在 `hugo.toml` 里设置 `theme = "ananke"`。
